## ğŸŒ Cara Deploy ke Streamlit Cloud (+10 Poin!)

### Langkah-langkah:

1. **Persiapan:**

   - Pastikan file `Tugas3.py` dan `requirements.txt` ada
   - Buat repository GitHub baru

2. **Upload ke GitHub:**

   ```bash
   git init
   git add .
   git commit -m "Initial commit - Dashboard Visualisasi"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO_NAME.git
   git push -u origin main
   ```

3. **Deploy:**

   - Buka [share.streamlit.io](https://share.streamlit.io)
   - Login dengan akun GitHub
   - Klik "New app"
   - Pilih repository, branch (main), dan file (Tugas3.py)
   - Klik "Deploy"

4. **Selesai!**
   - Tunggu beberapa menit
   - Dapatkan URL public untuk dishare
   - Contoh: `https://username-repo-name.streamlit.app`
